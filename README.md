# portfolio

Статический сайт‑портфолио **без библиотек, фреймворков и JavaScript** (чистые `HTML/CSS`).

## Файлы

- `index.html` — разметка и секции (Hero/Обо мне/Проекты/Навыки/Контакты)
- `styles.css` — адаптивные стили + авто‑тема через `prefers-color-scheme`

## Как запустить локально

Проще всего открыть `index.html` в браузере.

Если хотите запуск через локальный сервер (чтобы всё работало одинаково во всех браузерах):

```bash
python3 -m http.server 5173
```

Затем откройте `http://localhost:5173`.

## Где менять контент

- **Имя/описание**: `index.html` (секция `#top` и карточка справа)
- **Email/ссылки**: `index.html` (секция `#contact`, замените `you@example.com` и ссылки)
- **Проекты**: `index.html` (секция `#projects`, карточки проектов)
